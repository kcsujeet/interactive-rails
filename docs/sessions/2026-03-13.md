### Changes Made

#### 1. Custom Type 3 "Document Search Grid" Visualization (Level29Search.tsx)
- **Replaced PipelineFlow** with a custom "Document Search Grid" visualization
- **Observe phase**: Single posts table as a 100-block grid (20x5). When probe fires, red wave sweeps ALL blocks (sequential scan). Matched blocks turn green. Badges show: scan time, stemming status, ranking status
- **Reward phase**: Same grid returns. GIN Index Card appears (inverted index showing stemmed terms -> row IDs). Matching blocks go green instantly (no red wave). The contrast teaches what a GIN index does
- **Components created**: `DocumentGrid`, `GinIndexCard`, `NoIndexBanner`, `BlockedBanner`, `SearchBadges`, `SearchLegend`
- **Animation**: `runSeqScanAnimation` (observe, red wave), `runGinScanAnimation` (reward, instant match). `isAnimating` flag disables ProbeTerminal and StressTestPanel during animation
