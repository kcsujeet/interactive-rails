## Lessons Learned

- The codebase had THREE different slug patterns: `railsexpert` (one word), `rails-expert` (hyphenated), and `Rails Expert` (spaced). Each needed separate grep patterns to find.
- `bun install` does not always update `bun.lock` name fields when `package.json` names change. Deleting `bun.lock` and reinstalling forces a clean regeneration.
- The Edit tool requires reading a file in the current conversation context before editing, even if a subagent already read it.
