- `useRef` for `policyDeployedRef` avoids stale closures in movement interval

## Design Decisions
- Steps 0, 2 use TerminalChoiceStep for shell commands; steps 1, 3-6 use OptionCard for code choices (L12)
- Level 13 steps 0-2 are terminal, 3-5 are OptionCard
- Step types dispatched via `STEP_TYPES` array and `OPTION_STEP_CONFIG` record (no more offset arithmetic)
- Correct answer is never the first option (per CLAUDE.md)
- L12 uses "violet" color, L13 uses "emerald" color (testing theme). No color hints within a level
- Feedback explains why the wrong choice is wrong without revealing the correct answer
- Sequential phase flow: one thing at a time in the center panel, no split attention
- "Build the Fix" button uses `animate-in fade-in duration-500` (tw-animate-css per CLAUDE.md)
- Animation stops completely during build/activate phases (no wasted CPU)
