### Narrative Discovery: Account Model Never Introduced
While redesigning L34, discovered that the Account model used in the locking scenario was never introduced in any prior level. Further investigation revealed the Order model (used in Acts 5-8) was also never formally introduced.

## Major Decision: E-Commerce Narrative Reframe

Decided to switch the entire game narrative from social media (blog/posts) to e-commerce (products/orders).

**Why:** The social media narrative breaks down after Act 4. Acts 5-8 already reference Orders, Payments, Stripe, inventory, shipping, but those models were never introduced. The e-commerce narrative makes the full 56-level arc coherent.

**Impact Analysis:**
- 14 levels need no change (infrastructure, already e-commerce)
- 27 levels need rename (find-replace model names)
- 10 levels need rewrite (L8, L12, L18, L22, L30, L33, L36, L37, L43, L45)

**Core Model Mapping:**
- Post -> Product (name, description, price)
- Comment -> Review (body, rating)
- User -> User (same)
- Photo, Video -> ProductImage, ProductVideo
- Boost + CreditLog -> Order + OrderItem
- Account (phantom) -> removed
